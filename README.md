# 🎲 Bingo Game

A console-based Bingo game built in Python where players compete against a computer opponent. The game features randomized boards, turn-based gameplay, automatic bingo detection, and a save/load system that allows players to continue their progress at any time.

## ✨ Why This Project?

This project recreates the classic Bingo experience in a simple command-line environment while demonstrating important programming concepts such as file handling, game logic, data structures, and user interaction. It provides an engaging way to learn Python through a complete, interactive application.

## 🚀 Features

### 🎮 Gameplay

* Single-player mode against the computer
* Randomly generated 5×5 or 7×7 Bingo boards
* Turn-based number selection
* Automatic win detection
* Supports rows, columns, and diagonal bingo checks

### 💾 Save & Load System

* Save game progress at any time
* Continue previously saved games
* Stores boards, remaining numbers, and player information using file handling

### 🤖 Computer Opponent

* Random number selection
* Real-time board updates
* Automatic bingo tracking

## 🛠️ Technologies

* Python
* File Handling
* Functions
* Lists
* Random Module
* Console-Based Interface

## 📖 How to Use

1. Run the program.
2. Enter your name.
3. Choose:

   * **5** for a 5×5 board
   * **7** for a 7×7 board
4. Take turns selecting numbers from your board.
5. Enter **P** at any time to pause the game:

   * Continue playing
   * Save and exit
6. Complete **BINGO** before the computer to win.
